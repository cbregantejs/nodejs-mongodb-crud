
# CRUD NodeJs - MongoDB

Basic crud in nodejs using mongodb as database




## Features

- Upload images to cloudinary
- use mongoose
- use nodemon
- use babel



## Run Locally

Config enviroments in .env file

```bash
MONGODB_URI=
CLOUDINARY_URL=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
